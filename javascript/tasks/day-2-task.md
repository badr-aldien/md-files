# JavaScript Tasks - Day 2

---

## Task 1: Count 'e' Characters

- Accept a string from the user using `prompt()`.
- Count the number of times the letter **'e'** appears in the string.
- Display the count using `alert()` or `console.log()`.

---

## Task 2: Mirror Word Checker

- Ask the user to enter a word using `prompt()`.
- Ask the user (using a `confirm()` box) if they want to ignore uppercase and lowercase letters.
- Check if the word is a mirror word (looks the same forward and backward, like: `RADAR`, `NOON`, `MOOM`).

> **Note:** If the user chooses to ignore letter case, then `raDaR` is considered a mirror word.
